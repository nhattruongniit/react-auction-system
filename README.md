## Scripts

```bash

# run FE
$ cd auction-system-fe/

# install dependencies
$ yarn install

# create .env file
- copy .env.template

# run localhost
$ yarn dev

# run BE
$ cd auction-system-be/

# install dependencies
$ yarn install

# run localhost:3002
$ yarn dev
```
